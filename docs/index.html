Your Desktop AI Companion, Reinventing Dev Workflows

Office AI Agent is an open-source AI assistant running locally on your machine. Deeply integrated with powerful language models, it helps you complete varied development tasks like coding, debugging, and analysis through natural chat interactions and seamless tool invocation.

Key Features

Designed to boost developer productivity.

Intelligent Chat Interaction

Powered by the robust Gemini model, offering fluid, context-aware conversational experiences that accurately understand your development intent.

Powerful Toolset

Capable of reading/writing files, executing Shell commands, and searching codebases, seamlessly combining AI intelligence with your local development environment.

Code Understanding & Generation

Built-in code highlighting and Diff views make explaining old code, generating new modules, or reviewing changes intuitive and efficient.

Cross-Platform Desktop App

Built with Electron and React, providing a consistent, native experience across Windows, macOS, and Linux.

Flexible Process Management

Adopts a multi-process architecture to ensure AI tasks run in independent Workers, keeping the UI fluid and responsive without blocking your operations.

100% Open Source

Code is fully open and community-driven. We welcome your code contributions, suggestions, and collaborative efforts to build a stronger AI development tool.
