Transform CLI AI Agents into Modern Chat

AionUI is an open-source Electron application that wraps Claude Code, Google Gemini, and OpenAI Codex CLI agents in a beautiful, multi-user chat interface with enterprise SSO authentication and Docker deployment.

> The authentication system uses a dual-mode approach:
- AuthService.ts: JWT token management with 15-min access + 7-day refresh
- OidcService.ts: Enterprise SSO via OIDC (EntraID, Okta, etc.)
- TokenMiddleware.ts: Request validation with blacklist checking

Key Features

Enterprise-ready AI agent interface for development teams.

Multi-Agent Support

Seamlessly switch between Claude Code (ACP), Google Gemini, and OpenAI Codex agents in a unified interface.

Enterprise Authentication

OIDC/SSO integration with EntraID, Okta, and other providers. RBAC with admin/user roles and per-user data isolation.

MCP Tool Integration

Model Context Protocol support for extending agent capabilities with custom tools and server connections.

Channel Plugins

Connect AI agents to Telegram, Lark, and other messaging platforms via the extensible plugin system.

Docker Deployment

Production-ready Docker images with headless Electron, WebSocket real-time updates, and remote browser access.

LLM Gateway Support

Connect to LiteLLM, Azure OpenAI, Portkey, Kong AI Gateway, and other proxy providers for unified API access.

Get Started

Deploy AionUI in minutes with Docker or run locally.

1

Clone Repository

Clone the repository and navigate to the project directory.

git clone https://github.com/jrmatherly/aionui.git && cd aionui
→
2

Configure Environment

Copy the example environment file and add your API keys and OIDC configuration.

cp deploy/docker/.env.example deploy/docker/.env
→
3

Start Container

Build and start the Docker container. Access the UI at localhost:25808.

cd deploy/docker && docker-compose up -d
